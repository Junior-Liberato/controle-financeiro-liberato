# Controle Financeiro Liberato

Sistema Financeiro Familiar Liberato.

Projeto desenvolvido em HTML, CSS e JavaScript, com hospedagem prevista no GitHub Pages e dados sincronizados via Firebase.

## Objetivo

Criar uma central financeira familiar para controle de contas a pagar, receitas, cartões de crédito, faturas, histórico, relatórios e backup.
